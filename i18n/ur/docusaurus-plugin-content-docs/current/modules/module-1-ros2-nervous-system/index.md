# ماڈیول 1: روبوٹک اعصابی نظام (ROS 2)

جسمانی مصنوعی ذہانت اور انسانی روبوٹکس کی درسی کتاب کے ماڈیول 1 میں خوش آمدید۔ یہ ماڈیول ROS 2 کو "اعصابی نظام" کے طور پر متعارف کرواتا ہے جو روبوٹک نظاموں میں communication اور coordination کو ممکن بناتا ہے۔

## Overview

جیسے انسانی اعصابی نظام electrical signals کے ذریعے جسم کے functions کو coordinate کرتا ہے، اسی طرح ROS 2 مواصلاتی فریم ورک کے طور پر کام کرتا ہے جو روبوٹ کے مختلف components کو seamlessly مل کر کام کرنے کے قابل بناتا ہے۔ یہ ماڈیول آپ کو سمجھنے میں مدد کرے گا:

- ROS 2 کیا ہے اور یہ انسانی روبوٹکس کے لیے کیوں ضروری ہے
- Core ROS 2 concepts: nodes، topics، services، اور actions
- Communication patterns جو distributed robotic systems کو ممکن بناتے ہیں
- ROS 2 humanoid robot applications پر کیسے apply ہوتا ہے

## Learning Objectives

اس ماڈیول کے آخر تک، آپ قابل ہوں گے:

1. **ROS 2 کے کردار کی وضاحت** روبوٹکس میں بطور communication اور coordination framework
2. **مناسب communication patterns کی شناخت** (topics، services، actions) مختلف روبوٹک scenarios کے لیے
3. **ROS 2 components کو map کرنا** انسانی روبوٹ subsystems پر اور ان کے practical applications کو سمجھنا

## پیشگی ضروریات

اس ماڈیول کو شروع کرنے سے پہلے، آپ کے پاس ہونا چاہیے:

- **Python programming knowledge** - آپ Python syntax، functions، classes، اور basic programming concepts سے واقف ہوں
- **بنیادی روبوٹکس کے تصورات** - سمجھنا کہ روبوٹس کیا ہیں، basic components جیسے sensors اور actuators
